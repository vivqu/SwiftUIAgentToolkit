# Placeholder Images

Curated images for use in Swift prototypes. Drop images into the appropriate category folder and follow the naming convention below.

## Naming convention

| Category | Names | Count | Use for |
| -------- | ----- | ----- | ------- |
| Avatar / portrait (illustrated) | `avatar-illustrated-1.png` … `avatar-illustrated-6.png` | 6 | Profile photos, user avatars, social UI |
| Avatar / portrait (photo) | `avatar-1.jpg` … `avatar-6.jpg` | 6 | Realistic profile photos, social UI |
| Product | `product-1.jpg` … `product-10.jpg` | 10 | Product cards, shop listings |
| Hero / banner (nature) | `nature-1.jpg` … `nature-3.jpg` | 3 | Full-width headers, feature banners |
| Hero / banner (architecture) | `architecture-1.jpg` … `architecture-3.jpg` | 3 | Full-width headers, cityscapes |
| Abstract / background | `abstract-1.jpg` … `abstract-6.jpg` | 6 | Decorative backgrounds, cards |

## Rules

- **Format:** JPEG or PNG, square or landscape crops preferred
- **Resolution:** At least 600×600px so they look sharp on 3× displays

## Sources

- **Avatars (illustrated):** Illustrated personas from [Personas by Draftbit](https://personas.draftbit.com/)
- **Avatars (photo), Products, Heroes, Abstracts:** Photos from [Unsplash](https://unsplash.com/)

## Optimization

Images have been minified using [TinyPNG](https://tinypng.com/).

## Folders

```none
assets/placeholder-images/
├── avatars/      ← illustrated + photo portrait images
├── products/     ← product card images
├── heroes/       ← full-width banner images (nature + architecture subcategories)

```
